# IMC_basico 

O IMC é reconhecido como padrão internacional para avaliar o grau de sobrepeso e obesidade. 

É calculado dividindo o peso pela altura ao quadrado.

Calculo de IMC:

  IMC = PESO / ALTURA^2

Construido em linguagem Python
